
# ionic-auth-flow
Ionic auth flow and api consumption

# To run on the Browser
`ionic serve`


# Tech Used
- HTTPClientModule to consume API
- Storage and SQLite to persist data
- NavController for Routing
- Stand alone API Service and Storage Service
